# **üìò Vantage ‚Äî Frontend Development Document**

**Tech Stack:** React.js, SCSS, TypeScript (recommended), Zustand/Redux (for state), React Query (data fetching)
**Platforms:** Mobile (responsive) + Desktop
**Design Style:** Minimal FinTech UI with strong usability

---

# **1. Project Overview**

**Vantage** is a real-time StarkNet wallet intelligence dashboard + gamified rewards layer.
The frontend will:

* Display wallet analytics
* Show real-time events
* Enable swapping/sending
* Track DeFi positions
* Surface airdrop opportunities
* Run a leaderboard system
* Show NFT discovery & alerts

Frontend reads all required data from APIs / backend services (not included in this doc).

---

# **2. Frontend Architecture**

### **2.1 Framework & Libraries**

* **React.js**
* **SCSS Modules** (per-component styling)
* **React Router** (page navigation)
* **React Query** (live data & server state)
* **Zustand** or Redux Toolkit (global UI state: wallet, theme, toasts)
* **Chart.js or Recharts** (PNL graphs)
* **Framer Motion** (micro animations, optional)

### **2.2 Layout System**

* **Responsive grid** using CSS flex/grid
* **Mobile-first** approach
* Max width for desktop: **1280px**

### **2.3 Shared Components**

* Header/Navbar
* Wallet Connect Modal
* Token Selector Modal
* Table Components
* Tabs / Filters
* Toast Notifications
* Cards & Panels
* Pagination
* Icons
